# 数据采集指南 v3

## 目标
采集 beam 节奏型组合，训练节奏型分类器

---

## 一、节奏型采集（重点！）

### 采集要求

每种节奏型需要采集 **多种斜率**：
- 水平 (同音高，如 C4-C4)
- 上行斜率 (如 C4-E4, C4-G4)
- 下行斜率 (如 E4-C4, G4-C4)

**重要**: 每个节奏型画完整！包括符头+符干+beam

---

### 1. beam_8_8 - 采集 50 个

**两个八分音符 (1条beam)**

```
水平:        上行:        下行:
●═══●       ●═══●           ●═══●
│   │       │     │       │   │
```

采集分配:
- 水平: 15个
- 上行: 18个 (轻微~明显)
- 下行: 17个 (轻微~明显)

## 二、斜率参考

```
最大斜率约 30°:

    轻微上行 (~10°):          明显上行 (~25°):
    ●═══●                         ●
    │     │                      ╱│
                               ●══
                               │

    轻微下行 (~10°):          明显下行 (~25°):
        ●═══●                 ●
       │   │                  │══╲
                                  ●
                                  │
```

**不要超过 30° 斜率** - 太斜就像符干了

## 四、画法要点

### 笔画顺序（建议）
1. 先画符头 (从左到右)
2. 再画符干 (从上到下或从下到上)
3. 最后画beam (横向连接)

### 注意事项
1. **完整画** - 每个节奏型包含所有组件
2. **自然手写** - 正常书写速度
3. **符干方向一致** - 同一组内全上或全下
4. **beam清晰连接** - 要明显接到符干
5. **双beam间距** - 十六分音符的两条beam要能区分

### 不要
- 不要画超过30°的斜率
- 不要漏画符干
- 不要把beam画得太短（要连到符干）

    }
