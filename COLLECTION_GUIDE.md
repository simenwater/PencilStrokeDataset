## 一、单组件补充

### 1. clef (谱号) - 采集 25 个

```
高音谱号 (treble-clef):

    ╭─╮
    │ │
  ──┼─┼──
    │╭╯
  ──┼┼───
    ││
  ──┼╯───
    │
    ╰
```

**要求**：
- 主要采集高音谱号
- 大小适中，不要太小
- 可以有轻微变形

### 2. flag (旗帜) - 采集 35 个

```
向上旗帜 (flag-up):     向下旗帜 (flag-down):

    │                        │
    │                        │
    │╲                      ╱│
    │ ╲                    ╱ │
       ╲                  ╱
```

**要求**：
- 向上、向下各一半
- 单独画旗帜部分（弯钩）
- 八分音符旗帜为主

## 二、Beam 节奏型组合（新类别）

### 1. beam_8_8 - 采集 40 个

**两个八分音符连梁**

```
    ●     ●
    │     │
    ═══════    ← 1条beam
```

**要求**：
- 画完整的两个音符 + beam
- 音高可以变化（beam 可以有斜率）
- 符干方向一致

### 2. beam_16_16 - 采集 40 个

**两个十六分音符连梁**

```
    ●     ●
    │     │
    ═══════    ← 2条beam
    ═══════
```

**要求**：
- 两条 beam 连接
- 间距要看得出是两层

### 3. beam_16_16_8 - 采集 35 个

**前十六后八（常见节奏型）**

```
    ●  ●     ●
    │  │     │
    ══════════    ← 主beam（全连）
    ════          ← 次beam（只连前两个）
```

**要求**：
- 主 beam 连接全部
- 次 beam 只连接前两个十六分音符
- 第三个是八分音符（只有一条 beam）

### 4. beam_8_16_16 - 采集 35 个

**前八后十六（常见节奏型）**

```
    ●     ●  ●
    │     │  │
    ══════════    ← 主beam（全连）
          ════    ← 次beam（只连后两个）
```

**要求**：
- 主 beam 连接全部
- 次 beam 只连接后两个十六分音符
- 第一个是八分音符

### 5. beam_16x4 - 采集 30 个

**四个十六分音符连梁**

```
    ●  ●  ●  ●
    │  │  │  │
    ════════════    ← 2条beam
    ════════════
```

**要求**：
- 四个音符
- 两条完整的 beam

### 6. beam_triplet - 采集 25 个

**三连音（八分音符）**

```
       3
    ╭──┴──╮
    ●  ●  ●
    │  │  │
    ══════════    ← 1条beam
```

**要求**：
- 三个音符 + 一条 beam
- 上方可以画"3"标记（可选）
- 表示三连音节奏

## 四、采集注意事项

### 通用要求
1. **自然手写**：正常书写速度，不用刻意工整
2. **大小适中**：符合实际乐谱比例
3. **音高变化**：beam 组合可以画不同音高，产生斜率变化
4. **符干方向**：同一组内符干方向一致（全上或全下）

### Beam 专项
1. **斜率变化**：C4-D4 几乎水平，C4-B4 有明显斜率，都要采集
2. **连接清晰**：beam 要明显连接到符干
3. **层数分明**：十六分音符的两条 beam 要能区分

### 不需要采集
- **dot（附点）**：规则检测（符头右侧小点）
- **barline（小节线）**：规则检测（穿过五线谱的竖线）
- **stem（符干）**：已有 223 个，足够
